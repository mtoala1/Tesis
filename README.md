El primer modelo corresponde a una CNN entrenada desde cero. Esta arquitectura está compuesta por múltiples capas convolucionales seguidas de capas de agrupamiento máximo “MaxPooling”, una capa de regularización “Dropout” y una capa completamente conectada con función de activación “Softmax” para la clasificación multiclase. Este modelo permite establecer una línea base de comparación al no utilizar conocimiento previo aprendido en otros conjuntos de datos. (La Tabla 2) muestra la secuencia de capas convolucionales, las dimensiones de salida y el número total de parámetros entrenables, lo cual permite evidenciar la complejidad estructural del modelo base utilizado como punto de comparación experimental.
